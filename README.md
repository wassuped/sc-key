# ScKey

Rust port of [ScKey](https://github.com/FMZNkdv/ScKey)

## Quick Start

### Using Pre-built Binaries (Recommended)
1. Download the latest release from the [Releases page](https://github.com/wassuped/sc-key/releases)
2. Run directly from command line:
   ```bash
   ./sc-key <obfuscated_key>
   ```

### Building from Source
```bash
git clone https://github.com/wassuped/sc-key
cd sc-key
cargo run <obfuscated_key>
```
